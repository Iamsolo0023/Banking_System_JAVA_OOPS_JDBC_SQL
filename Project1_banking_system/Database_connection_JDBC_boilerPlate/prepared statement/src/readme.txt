=======================================
        🏦 BANKING MANAGEMENT SYSTEM     
=======================================

🔹 Description:
   This is a terminal-based banking system where users can:
   - Register a new account
   - Login to their existing account
   - Perform banking transactions like:
     ✅ Deposit (Credit)
     ✅ Withdraw (Debit)
     ✅ Transfer funds
     ✅ Check balance
   - Secure logout mechanism

---------------------------------------
🔹 MAIN MENU:
---------------------------------------
1️⃣ Register
2️⃣ Login
3️⃣ Exit

---------------------------------------
🔹 AFTER LOGIN - ACCOUNT OPERATIONS:
---------------------------------------
1️⃣ Debit Money
2️⃣ Credit Money
3️⃣ Transfer Money
4️⃣ Check Balance
5️⃣ Log Out
